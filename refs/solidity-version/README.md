# Install Contracts Dependencies

```
pnpm i
```

# Compile Contracts

```
pnpm compile
```

# Test Contracts

```
pnpm test
```

# Deploy Contracts

```
pnpm check-in:deploy --network <>
pnpm vault:deploy --network <>
pnpm swap-handler:deploy --network <>
```

# Call Contracts

```
pnpm check-in:call --network <>
pnpm vault:call --network <>
pnpm swap-handler:call --network <>
```

# Verify Contracts

```
pnpm check-in:verify --network <>
pnpm vault:verify --network <>
pnpm swap-handler:verify --network <>
```
